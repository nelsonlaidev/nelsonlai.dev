## Quién soy

Soy un Ingeniero Full Stack en Hong Kong. Tengo 19 años y estoy aprendiendo TypeScript, Next.js, React.js, Node.js y gestión de bases de datos. Ahora tengo pasión por aprender UI/UX.

Empecé a aprender desarrollo web el 5 de diciembre de 2020 (tenía 14 años). Es un gran desafío porque es solo uno de mis pasatiempos y aún no he tomado ningún curso. Aprendo de forma autodidacta en YouTube. Uno de mis canales favoritos es [The Net Ninja](https://www.youtube.com/@NetNinja). Aprendí muchas habilidades de desarrollo web de él.

Me encanta usar [Next.js](https://nextjs.org/) para crear un sitio web. Luego uso [GitHub](https://github.com) para alojar mi código. Después, uso [Vercel](https://vercel.com) para desplegar mi sitio web.

## Acerca de este sitio

- Framework: [Next.js](https://nextjs.org/)
- Base de datos: [PostgreSQL](https://www.postgresql.org/)
- ORM: [Drizzle](https://orm.drizzle.team/)
- Autenticación: [Better Auth](https://www.better-auth.com/)
- Despliegue: [Vercel](https://vercel.com)
- Contenido: [MDX](https://mdxjs.com/)
- Analíticas: [Umami](https://github.com/umami-software/umami)
- Estilos: [Tailwindcss](https://tailwindcss.com)
- Resaltado de sintaxis: [Shiki](https://github.com/shikijs/shiki)

Gracias a **Timothy Lin** por el
[tailwind-nextjs-starter-blog](https://github.com/timlrx/tailwind-nextjs-starter-blog). Me ayudó cuando no tenía idea de cómo crear un blog desde cero. Mi código no es como el código inicial porque cambié mucho :)
