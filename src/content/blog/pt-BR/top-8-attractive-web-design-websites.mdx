## Prefácio

Recentemente, juntei uma coleção excepcional de sites com design de UI/UX que me deixaram admirado após minha exploração inicial. Acredito que esses sites possam enriquecer muito suas habilidades em UI. Aqui está uma lista dos meus 8 sites favoritos (sem ordem particular).

## Raycast

O Raycast é uma ferramenta de produtividade que permite a conclusão rápida de tarefas. Sua página inicial me cativou com seu design de UI, com minhas cores gradientes favoritas e vários tons de vermelho (a cor do tema).

Ao rolar para baixo, exibem-se seus recursos exclusivos, o que demonstra claramente os recursos do produto. Isso me levou a baixar prontamente o software depois de comprar um [MacBook Air de 15"](/uses). O esforço empenhado na criação de imagens, design de UI e seleção de cores por seus desenvolvedores é evidente no site e vale a pena aprender.

## Linear

Linear é uma poderosa ferramenta de gerenciamento de projetos projetada para designers e equipes. Ela oferece uma interface intuitiva e uma variedade de recursos para organizar e acompanhar projetos de design. Seja criando novos produtos, atualizando os existentes, lidando com feedback ou tarefas, o Linear fornece ferramentas colaborativas e fluxos de trabalho para garantir execução eficiente de projetos. Através do Linear, equipes de design podem colaborar melhor, acompanhar o progresso e garantir gerenciamento eficaz em cada etapa.

As animações e o uso de introduções de recursos do produto em estilo de blocos na página inicial são envolventes, e o efeito de fundo desfocado é cativante.

## Figma

O Figma não precisa de introdução — é um software profissional de design UI/UX, e seu site é igualmente impressionante. Seja para pequenos projetos ou grandes equipes, o Figma facilita que designers alcancem criatividade enquanto mantêm fluxos de trabalho colaborativos suaves.

O site emprega várias animações, e sua página inicial é atraente e funcional, oferecendo um ponto de entrada intuitivo para os usuários entenderem o valor do Figma.

## Next.js

Next.js é um framework React popular, projetado para o desenvolvimento de sites e aplicações modernas. Ele prioriza desempenho e escalabilidade, oferecendo recursos poderosos de renderização no servidor e geração de sites estáticos que otimizam a velocidade de carregamento e os índices de SEO. Seja para criar aplicações dinâmicas ou sites com conteúdo estático, o Next.js simplifica os processos de desenvolvimento e oferece experiências de usuário superiores.

O design minimalista do site oficial do Next.js é sofisticado, e a parte mais interessante é a seção "Powered By" — uma "CPU" criada usando CSS, surpreendente mas eficaz.

## PlanetScale

O PlanetScale é uma plataforma inovadora focada no gerenciamento de bancos de dados. Ele oferece ferramentas e funcionalidades modernas que ajudam equipes de desenvolvimento a projetar, implantar e manter bancos de dados de forma eficaz. A singularidade do PlanetScale está na sua facilidade de uso e capacidades de automação, aliviando o fardo do gerenciamento de bancos de dados e permitindo que os desenvolvedores se concentrem em criar aplicações de alta qualidade.

O uso de gradientes, imagens ricas e ilustrações no site é atraente, e minha parte favorita é o "Tacômetro" no meio. Uma análise das ferramentas de desenvolvimento revela como eles usaram CSS para criar esse efeito intrigante.

## Fig

O Fig é uma ferramenta de linha de comando que melhora significativamente a eficiência com autocomplete e escrita simplificada de scripts, otimizando fluxos de trabalho. Ele possui uma biblioteca de plugins poderosa e rica onde os usuários podem instalar vários plugins convenientes.

O site emprega vários elementos de gradiente, capturando a atenção do usuário de forma eficaz.

## Pipe

Pipe é uma plataforma de capital moderna que permite que empresas convertam receita futura em capital antecipado. Conexões de dados seguras e em tempo real simplificam o processo de obtenção de capital operacional para empresas. Não há diluição ou obstáculos de financiamento tradicionais. Seu capital disponível depende inteiramente de sua receita e saúde empresarial. Basta registrar, conectar e acessar fundos através da Pipe.

A página inicial apresenta efeitos especiais impressionantes, e o design de imagem, tamanho da fonte e interface são visualmente atraentes.

## Framer

Framer é uma poderosa ferramenta de design interativo e prototipagem criada para designers e desenvolvedores. Ele oferece uma interface intuitiva e uma riqueza de elementos interativos, permitindo que os usuários criem protótipos altamente interativos e designs de interface sem esforço. O Framer integra vários recursos, incluindo animações, gestos, micro-interações e edição de código, permitindo que os usuários concretizem criativamente seus conceitos de design. Seja para aplicações móveis ou design web, o Framer fornece ferramentas excepcionais para ajudar os usuários a criar experiências interativas cativantes.

As seções interativas no site, juntamente com o esquema de cores e animações, refletem um alto nível de profissionalismo.
