## Prefacio

Recientemente reuní una colección de sitios web excepcionales en diseño UI/UX que me dejaron asombrado en mi primera exploración. Creo que estos sitios web pueden enriquecer mucho tus habilidades de UI. Aquí está la lista de mis 8 sitios web favoritos (sin orden particular).

## Raycast

Raycast es una herramienta de productividad que permite completar tareas rápidamente. Su página de inicio me cautivó con su diseño de UI, que presenta mis colores degradados favoritos y varios tonos de rojo (el color temático).

Al desplazarse hacia abajo se muestran sus características únicas, demostrando claramente las capacidades del producto. Esto me llevó a descargar el software de inmediato después de comprar un [MacBook Air 15"](/uses). El esfuerzo puesto en la creación de imágenes, el diseño de UI y la selección de colores por parte de sus desarrolladores es evidente en el sitio web y vale la pena aprender de ello.

## Linear

Linear es una poderosa herramienta de gestión de proyectos diseñada para diseñadores y equipos. Ofrece una interfaz intuitiva y una variedad de funciones para organizar y rastrear proyectos de diseño. Ya sea diseñando nuevos productos, actualizando los existentes, gestionando comentarios o tareas, Linear proporciona herramientas de colaboración y flujos de trabajo para garantizar una ejecución eficiente del proyecto. A través de Linear, los equipos de diseño pueden colaborar mejor, rastrear el progreso y asegurar una gestión efectiva en cada etapa.

Las animaciones y el uso de introducciones de características de productos en estilo mosaico en la página de inicio son atractivos, y el efecto de fondo desenfocado es cautivador.

## Figma

Figma no necesita presentación: es un software profesional de diseño UI/UX, y su sitio web es igualmente impresionante. Ya sea para proyectos pequeños o grandes equipos, Figma facilita a los diseñadores lograr la creatividad mientras mantienen flujos de trabajo de colaboración fluidos.

El sitio web emplea diversas animaciones, y su página de inicio es tanto atractiva como funcional, ofreciendo un punto de entrada intuitivo para que los usuarios comprendan el valor de Figma.

## Next.js

Next.js es un popular framework de React diseñado para el desarrollo de sitios web y aplicaciones modernas. Enfatiza el rendimiento y la escalabilidad, proporcionando potentes capacidades de renderizado del lado del servidor y generación de sitios web estáticos que optimizan la velocidad de carga de páginas web y el rendimiento SEO. Ya sea para construir aplicaciones dinámicas o sitios web de contenido estático, Next.js simplifica los procesos de desarrollo y ofrece experiencias de usuario superiores.

El diseño minimalista del sitio web oficial de Next.js es sofisticado, y la parte más interesante es la sección "Powered By", una "CPU" creada con CSS, sorprendente pero efectiva.

## PlanetScale

PlanetScale es una plataforma innovadora enfocada en la gestión de bases de datos. Ofrece herramientas y funcionalidades modernas que ayudan a los equipos de desarrollo a diseñar, desplegar y mantener bases de datos de manera efectiva. La singularidad de PlanetScale radica en su facilidad de uso y capacidades de automatización, aliviando la carga de la gestión de bases de datos y permitiendo a los desarrolladores centrarse en crear aplicaciones de alta calidad.

El uso de degradados, imágenes ricas e ilustraciones en el sitio web es atractivo, y mi parte favorita es el "Tacómetro" en el centro. Una mirada a las herramientas de desarrollo revela cómo usaron CSS para crear este intrigante efecto.

## Fig

Fig es una herramienta de línea de comandos que mejora significativamente la eficiencia con autocompletado y escritura simplificada de scripts, optimizando los flujos de trabajo. Cuenta con una biblioteca de plugins potente y rica donde los usuarios pueden instalar varios plugins convenientes.

El sitio web emplea numerosos elementos de degradado, captando la atención de los usuarios de manera efectiva.

## Pipe

Pipe es una plataforma de capital moderna que permite a las empresas convertir ingresos futuros en capital inmediato. Las conexiones de datos seguras y en tiempo real simplifican el proceso de obtención de capital operativo para las empresas. No hay dilución ni obstáculos de financiamiento tradicional. Tu capital disponible depende enteramente de tus ingresos y la salud del negocio. Simplemente regístrate, conecta y accede a fondos a través de Pipe.

La página de inicio muestra impresionantes efectos especiales, y el diseño de imágenes, el tamaño de la fuente y la interfaz son visualmente atractivos.

<LinkCard href='https://pipe.com' hostname='pipe.com' title='Pipe' />

## Framer

Framer es una poderosa herramienta de diseño interactivo y prototipado creada para diseñadores y desarrolladores. Ofrece una interfaz intuitiva y una gran cantidad de elementos interactivos, lo que permite a los usuarios crear prototipos y diseños de interfaz altamente interactivos sin esfuerzo. Framer integra diversas funciones, incluidas animaciones, gestos, microinteracciones y edición de código, lo que permite a los usuarios realizar creativamente sus conceptos de diseño. Ya sea para aplicaciones móviles o diseño web, Framer proporciona herramientas excepcionales para ayudar a los usuarios a crear experiencias interactivas cautivadoras.

Las secciones interactivas del sitio web, junto con la paleta de colores y las animaciones, reflejan un alto nivel de profesionalismo.
